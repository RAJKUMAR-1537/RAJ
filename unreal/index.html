ID: #01 ENVIRONMENT DESIGN

GREEN LAND

Mission Brief

Created a real-time cinematic forest environment using Unreal Engine foliage system with instanced vegetation for performance optimization. Implemented dynamic lighting setup with directional light, global illumination, and post-process color grading to achieve cinematic mood. Used PBR materials, LOD optimization, and cinematic camera movement to enhance realism and visual storytelling.
